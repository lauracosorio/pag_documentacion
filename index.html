Introducción

CSS (Cascading Style Sheets) es el código que usa para diseñar su página web. Los conceptos básicos de CSS lo llevan a través de lo que necesita para comenzar. Responderemos preguntas como: ¿Cómo hago que mi texto sea negro o rojo? ¿Cómo hago para que mi contenido aparezca en tal o cual lugar en la pantalla? ¿Cómo decoro mi página web con imágenes de fondo y colores?

¿Qué es CSS?

Al igual que HTML, CSS no es realmente un lenguaje de programación. Tampoco es un lenguaje de marcado , es un lenguaje de hoja de estilo . Esto significa que le permite aplicar estilos selectivamente a elementos en documentos HTML. Por ejemplo, para seleccionar todos los elementos de párrafo en una página HTML y convertir el texto dentro de ellos en rojo, escribiría este CSS:


p { color: red; }

Probémoslo: pegue esas tres líneas de CSS en un nuevo archivo en su editor de texto, y luego guarde el archivo como style.cssen su stylesdirectorio.
Pero aún necesitamos aplicar el CSS a su documento HTML. De lo contrario, el estilo CSS no afectará la forma en que su navegador muestra el documento HTML. (Si no ha seguido con nuestro proyecto, lea Manejo de archivos y conceptos básicos de HTML para averiguar qué debe hacer primero).

1. Abra su index.htmlarchivo y pegue la siguiente línea en algún lugar de la cabeza:

link href="styles/style.css" rel="stylesheet"

2. Guárdelo index.htmly cárguelo en su navegador.

Anatomía de un conjunto de reglas CSS

Veamos el CSS anterior con un poco más de detalle:



Toda la estructura se llama un conjunto de reglas (pero a menudo "regla" para abreviar). Tenga en cuenta los nombres de las partes individuales:

Selector

El nombre del elemento HTML al comienzo del conjunto de reglas. Selecciona los elementos que se diseñarán (en este caso, elementos p). Para diseñar un elemento diferente, simplemente cambie el selector.

Declaración

Una sola regla, como color: red; especifica qué propiedades del elemento desea diseñar.

Propiedades

Formas en que puede diseñar un elemento HTML dado. (En este caso, color es una propiedad de los elementos p ). En CSS, usted elige qué propiedades desea afectar en su regla.

El valor de la propiedad

A la derecha de la propiedad, después de los dos puntos, tenemos el valor de la propiedad , que elige uno de los muchos aspectos posibles para una propiedad determinada (hay muchos valores color además red).

Tenga en cuenta las otras partes importantes de la sintaxis:

· Cada conjunto de reglas (aparte del selector) debe estar envuelto en llaves ({}) .
· Dentro de cada declaración, debe usar dos puntos (:) para separar la propiedad de sus valores.
· Dentro de cada conjunto de reglas, debe usar un punto y coma(;) para separar cada declaración de la siguiente.

Entonces, para modificar varios valores de propiedad a la vez, solo necesita escribirlos separados por punto y coma, de esta manera:

p { color: red; width: 500px; border: 1px solid black; }

Seleccionar múltiples elementos

También puede seleccionar varios tipos de elementos y aplicar un único conjunto de reglas a todos ellos. Múltiples selectores están separados por comas. Por ejemplo:


p, li, h1 { color: red; }

Fuentes y texto

Ahora que hemos explorado algunos conceptos básicos de CSS, comencemos a agregar más reglas e información a nuestro archivo style.css para que nuestro ejemplo se vea bien. Comencemos haciendo que nuestras fuentes y texto se vean un poco mejor.

  1. En primer lugar, regrese y encuentre la salida de Google Fonts que almacenó en un lugar seguro. Agregue el link elemento en algún lugar dentro de la cabeza de su index.html . Se verá más o menos así:

    2. link href="https://fonts.googleapis.com/css?family=Open+Sans" rel="stylesheet"

  2. Este código vincula su página a una hoja de estilo que descarga la familia de fuentes Open Sans junto con su página web y le permite configurarla en sus elementos HTML utilizando su propia hoja de estilos.
    Luego, elimine la regla existente que tiene en su archivo style.csss. Fue una buena prueba, pero el texto rojo no se ve muy bien.

  3. Agregue las siguientes líneas en su lugar, reemplazando la línea de marcador de posición con la línea font-family real que obtuvo de Google Fonts. ( font-family solo significa la (s) fuente (s) que desea usar para su texto). Esta regla primero establece una fuente base global y un tamaño de fuente para toda la página (ya que html es el elemento padre de toda la página, y todos los elementos dentro de ella heredan el igual font-size y font-family):

    4. html { font-size: 10px; font-family: }

    Nota : Cualquier cosa en un documento CSS entre /*y */es un comentario CSS , que el navegador ignora cuando representa el código. Este es un lugar para escribir notas útiles sobre lo que está haciendo.
  4. hora vamos a establecer los tamaños de fuente para los elementos que contienen texto, en el interior del cuerpo HTML ( h1, li y p). También centraremos el texto de nuestro encabezado y estableceremos un poco de altura de línea y espacio entre letras en el contenido del cuerpo para que sea un poco más legible:

    h1 { font-size: 60px; text-align: center; }

    p, li {font-size: 16px; line-height: 2; letter-spacing: 1px; }

    Puede ajustar estos px a lo que quiera para que su diseño se vea como lo desea

Cajas, cajas, se trata de cajas

Una cosa que notará al escribir CSS es que gran parte se trata de cuadros: establecer su tamaño, color, posición, etc. La mayoría de los elementos HTML en su página pueden considerarse cuadros que se encuentran uno encima del otro.

No es sorprendente que el diseño CSS se base principalmente en el modelo de caja. Cada uno de los bloques que ocupan espacio en su página tiene propiedades como esta:

  • padding, el espacio justo alrededor del contenido (por ejemplo, alrededor del texto del párrafo).
  • border, la línea continua que se encuentra justo fuera del relleno.
  • margin, el espacio alrededor del exterior del elemento.

En esta sección también usamos:

  • width (de un elemento).
  • background-color, el color detrás del contenido y el relleno de un elemento.
  • color, el color del contenido de un elemento (generalmente texto).
  • text-shadow: establece una sombra paralela en el texto dentro de un elemento.
  • display: establece el modo de visualización de un elemento (no se preocupe por esto todavía).
¡Entonces, comencemos y agreguemos más CSS a nuestra página! Siga agregando estas nuevas reglas al final de la página y no tenga miedo de experimentar con los valores cambiantes para ver cómo resulta.

Cambiar el color de la página


html { background-color: #00539F; }

Esta regla establece un color de fondo para toda la página. Cambie el código de color anterior al color que elija al planificar su sitio .

Ordenar el cuerpo


body { width: 600px; margin: 0 auto; background-color: #FF9500; padding: 0 20px 20px 20px; border: 5px solid black; }

Ahora para el elemento body. Aquí hay bastantes declaraciones, así que analicémoslas una por una:

  • width: 600px; - esto obliga al cuerpo a tener siempre 600 píxeles de ancho.
  • margin: 0 auto;- Cuando establece dos valores en una propiedad como margino padding, el primer valor afecta el lado superior e inferior del elemento ( 0en este caso) y el segundo valor el lado izquierdo y derecho (aquí, autoes un valor especial que divide el valor disponible espacio horizontal uniformemente entre izquierda y derecha). También puede usar uno, tres o cuatro valores, como se documenta aquí .
  • background-color: #FF9500; - como antes, esto establece el color de fondo del elemento. Hemos usado una especie de naranja rojizo para el cuerpo en lugar del azul oscuro para el elemento, pero no dude en seguir adelante y experimentar.
  • padding: 0 20px 20px 20px; - tenemos cuatro valores establecidos en el relleno, para hacer un poco de espacio alrededor de nuestro contenido. Esta vez no estamos configurando ningún relleno en la parte superior del cuerpo, y 20 píxeles a la izquierda, abajo y derecha. Los valores se establecen arriba, derecha, abajo, izquierda, en ese orden. Al igual que con margin, también puede usar uno, dos o tres valores, como se documenta en la sintaxis de relleno .
  • · border: 5px solid black; - esto toma valores para el ancho, el estilo y el color del borde, en nuestro caso simplemente configurando un borde negro sólido de 5 píxeles de ancho en todos los lados del cuerpo.

Posicionar y diseñar el título de nuestra página principal

h1 { margin: 0; padding: 20px 0; color: #00539F; text-shadow: 3px 3px 1px black; }

Es posible que hayas notado que hay una brecha horrible en la parte superior del cuerpo. Eso sucede porque los navegadores aplican un estilo predeterminado al elemento h1 (entre otros), ¡incluso cuando no ha aplicado ningún CSS en absoluto! Puede parecer una mala idea, pero queremos que incluso una página web sin estilo tenga una legibilidad básica. Para deshacernos de la brecha, anulamos el estilo predeterminado configurando
margin: 0;.

A continuación, configuramos el relleno superior e inferior del encabezado en 20 píxeles e hicimos que el texto del encabezado tenga el mismo color que el color de fondo HTML.

Una propiedad bastante interesante que hemos usado aquí es la text-shadow que aplica una sombra de texto al contenido de texto del elemento. Sus cuatro valores son los siguientes:

  • El primer valor de píxel establece el desplazamiento horizontal de la sombra del texto: qué tan lejos se mueve: un valor negativo debe moverlo hacia la izquierda.
  • El segundo valor de píxel establece el desplazamiento vertical de la sombra del texto: en este ejemplo, cuánto se mueve hacia abajo; un valor negativo debería moverlo hacia arriba.
  • El tercer valor de píxel establece el radio de desenfoque de la sombra; un valor mayor significará una sombra más borrosa.
  • El cuarto valor establece el color base de la sombra.
¡De nuevo, intente experimentar con diferentes valores para ver qué se le ocurre!

Centrando la imagen


img { display: block; margin: 0 auto; }

Finalmente, centraremos la imagen para que se vea mejor. Podríamos usar el margin: 0 autotruco nuevamente como lo hicimos anteriormente para el cuerpo, pero también tenemos que hacer algo más. El elemento body está a nivel de bloque , lo que significa que ocupa espacio en la página y puede tener margen y otros valores de espaciado aplicados. Las imágenes, por otro lado, son elementos en línea , lo que significa que no pueden. Entonces, para aplicar márgenes a la imagen, tenemos que dar el comportamiento de nivel de bloque de imagen usando display: block;.

Nota : Las instrucciones anteriores suponen que está utilizando una imagen más pequeña que el ancho establecido en el cuerpo (600 píxeles). Si su imagen es más grande, desbordará el cuerpo y se extenderá al resto de la página. Para rectificar esto, puede 1) reducir el ancho de la imagen usando un editor de gráficos , o 2) ajustar el tamaño de la imagen usando CSS configurando la propiedad width en el elemento img con un valor menor (por ejemplo, 400 px;).
